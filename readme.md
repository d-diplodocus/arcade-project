# Running Robot Boy

### Created by Michael Roberts and Dezirae Choi

You play as Robot Boy, running as fast as he can! Collect stars and watch out for enemies! Bounce on their heads!

### Controls

Left/Right Arrow Keys: Move left/right\
Up/Space Bar: Jump
